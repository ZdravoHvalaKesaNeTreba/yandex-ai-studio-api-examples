# 🎙️ Веб-интерфейс голосового агента

Современный веб-интерфейс для голосового ассистента с дизайном в стиле Яндекс.

## ✨ Особенности

- 🎨 Минималистичный дизайн в стиле Яндекс Старт
- 📱 Полностью адаптивный (desktop, tablet, mobile)
- 🎤 Голосовое взаимодействие в реальном времени
- 🔊 Воспроизведение ответов ассистента
- ⚡ WebSocket для low-latency коммуникации
- 🌊 Плавные анимации точек при прослушивании/разговоре

## 🚀 Быстрый старт

### 1. Убедитесь, что настроены креденшалы

Файл `.env` должен находиться в директории `realtime/`:

```bash
YANDEX_CLOUD_FOLDER_ID=ваш_folder_id
YANDEX_CLOUD_API_KEY=ваш_api_key
```

### 2. Запустите веб-сервер

```bash
cd realtime/web
bash start_web.sh
```

### 3. Откройте в браузере

Перейдите по адресу: `http://localhost:5000`

## 📱 Использование

1. **Разрешите доступ к микрофону** при первом запросе браузера
2. **Нажмите кнопку микрофона** внизу экрана
3. **Начните говорить** - ваша речь будет распознаваться в реальном времени
4. **Дождитесь ответа** - ассистент ответит голосом
5. **Нажмите кнопку стоп** для завершения сессии

## 🎨 Дизайн

Интерфейс вдохновлен дизайн-системой Яндекса:

- **Цвета**: черный (#000000), желтый (#FFCC00), красный (#FF3333)
- **Шрифты**: системные (San Francisco на iOS, Roboto на Android)
- **Анимации**: плавные переходы с cubic-bezier
- **Адаптивность**: от 360px до desktop

## 🏗️ Архитектура

### Backend (Flask + SocketIO)

- [`app.py`](app.py) - основной сервер
- WebSocket для двусторонней коммуникации
- Асинхронная работа с Yandex Realtime API

### Frontend

- [`templates/index.html`](templates/index.html) - HTML структура
- [`static/css/style.css`](static/css/style.css) - стили и анимации
- [`static/js/main.js`](static/js/main.js) - логика взаимодействия

## 🔧 Технологии

**Backend:**
- Flask 3.0+
- Flask-SocketIO 5.3+
- aiohttp 3.9+

**Frontend:**
- Vanilla JavaScript (ES6+)
- Socket.IO Client 4.5+
- Web Audio API
- MediaRecorder API

## 🌐 Доступ с мобильного

### Локальная сеть

1. Узнайте IP адрес вашего компьютера:
   ```bash
   # macOS/Linux
   ifconfig | grep "inet "
   # Windows
   ipconfig
   ```

2. На мобильном откройте:
   ```
   http://ВАШ_IP:5000
   ```

### Туннелирование (ngrok)

Для доступа из интернета:

```bash
# Установите ngrok
brew install ngrok

# Запустите туннель
ngrok http 5000
```

Используйте предоставленный URL на любом устройстве.

## 🐛 Troubleshooting

### Микрофон не работает

- Проверьте разрешения в браузере
- Используйте HTTPS (или localhost)
- На iOS используйте Safari (Chrome не поддерживает Web Audio на iOS)

### Нет звука

- Проверьте громкость устройства
- Убедитесь, что не включен беззвучный режим
- Попробуйте другой браузер

### Ошибка аутентификации

- Проверьте креденшалы в `.env` файле
- Убедитесь, что API ключ валиден
- Проверьте, что folder_id корректный
